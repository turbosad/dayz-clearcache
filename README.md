## 🛠 Usage
1. Keep `ClearGPU.ps1` and `ClearGPU.bat` in the same folder.
2. Run `ClearGPU.bat` to initiate the cleanup with Admin rights.
3. Check your Desktop for `GPU_Cleanup_Log.txt` if any errors occur.
